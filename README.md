# Secure Messenger Backend

Производительный и безопасный сервер для мессенджера со сквозным шифрованием (E2EE), написанный на Go.

## Технологический стек
- **Язык**: Go 1.25+
- **Фреймворки**: Gin (HTTP REST), Gorilla WebSocket (Real-time)
- **База данных**: PostgreSQL (pgx)
- **Инфраструктура**: Docker, Nginx, Swagger UI

## Требования для развертывания
- Docker и Docker Compose (для контейнерного развертывания)
- Go 1.25+ (для локальной разработки без Docker)

## Инструкция по развертыванию (Production / Docker)

Самый простой и надежный способ запустить серверную часть со всеми зависимостями (БД, Swagger) — использовать Docker Compose.

1. **Клонирование репозитория:**
   ```bash
   git clone <url-репозитория>
   cd messenger-backend
   ```

2. **Настройка переменных окружения:**
   Скопируйте шаблонный файл `.env.example` в файл `.env` (он используется Docker Compose):
   ```bash
   cp .env.example .env
   ```
   *При необходимости откройте файл `.env` и отредактируйте параметры подключения к БД, секреты для JWT и другие настройки.*

3. **Сборка и запуск контейнеров:**
   Выполните команду для старта всех сервисов в фоновом режиме:
   ```bash
   docker-compose up -d --build
   ```

4. **Проверка статуса:**
   Убедитесь, что все контейнеры (app, db, redis, nginx, swagger-ui) успешно запущены:
   ```bash
   docker-compose ps
   ```

### Доступные сервисы после развертывания:
- **API (Nginx):** `http://localhost:80` (все запросы к `/api/v1` маршрутизируются в Go-приложение)
- **Swagger UI (Документация):** `http://localhost:8081` — интерактивная документация REST API.
